## Installation

1. Install the plugin using composer:
```shell
composer require synolia/orocommerce-stock-alert-plugin
```
2. Run the migrations
```shell
bin/console oro:migration:load --force
```
3. Clear cache
```shell
bin/console cache:clear
```
